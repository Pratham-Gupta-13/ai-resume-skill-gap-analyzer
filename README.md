AI Resume Skill Gap Analyzer

This project analyzes resume text using Natural Language Processing (NLP)
and compares it with job role requirements.

Features:
- Skill extraction using NLP
- Skill gap analysis
- Match percentage calculation
- Web interface using Flask

Technology Stack:
Python, Flask, NLTK, HTML, CSS
